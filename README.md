# OrangeHRM Playwright Tests

Small Playwright test suite for the OrangeHRM demo application.


## Tech
- Playwright
- TypeScript

## Setup
```bash
npm install
npm test
npm run test:headed
npm run test:ui
npm run test:debug
```

## Covered scenarios
- Login (happy path and invalid credentials)
- Sidebar navigation (PIM, Leave, Admin)
- Add employee (PIM)
- Logoutg
